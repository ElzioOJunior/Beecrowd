Projeto desenvolvido seguindo as orientações para o teste.

Caso necessite rodar o projeto localmente, alterar a string de conexão do banco de dados de db para localhost

De:
 "ConnectionStrings": {
   "DefaultConnection": "Host=db;Database=SalesApiDb;Username=postgres;Password=admin"
 }

Para:
 "ConnectionStrings": {
   "DefaultConnection": "Host=localhost;Database=SalesApiDb;Username=postgres;Password=admin"
 }
